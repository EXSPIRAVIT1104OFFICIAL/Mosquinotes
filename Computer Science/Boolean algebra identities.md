# Definition

# Notes

Identity:
- `1.A = A`
- `0 + A = A`

Null:
- `0.A = 0`
- `1 + A = 1`

Idempotent:
- `A.A = A`
- `A + A = A`

Inverse:
- `A.A' = 0`
- `A + A' = 1`

Commutative:
- `A.B = B.A`
- `A + B = B + A`

Associative:
- `(A.B).C = A.(B.C)`
- `(A + B) + C = A + (B + C)`

Distributive:
- `A + B.C = (A + B).(A + C)`
- `A.(B + C) = A.B + A.C`

Absorption:
- `A.(A + B) = A`
- `A + A.B = A`

De Morgan's:
- `(A.B)' = A' + B'`
- `(A + B)' = A'.B'`

Double complement:
- `A'' = A`
