# Definition

A [[Binary number]] obtained by subtracting each digit from one.

# Notes

For example:
`10100100`, `1` switched to `0` and `0` switched to `1`. The `10100100` is `01011011` in [[One's complement]] form, represents -91 in [[Denary number]].
